# Dog finder



## Opis aplikacji

Aplikacja wyszukująca rasy psów z API. Projekt jest zadaniem rekrutacyjnym dla Wakacje.pl

## Funkcje

- wyszukanie rasy psa z listy wszystkich ras
- wyszukanie rasy psa przez 



## Technologie 

* React.js
* React router dom
